CEIR Platform

Ruby on Rails PostgreSQL Apache Kafka Apache Spark Docker CI/CD

Central Equipment Identity Register platform processing national-scale device registration data across multiple mobile network operators. The system ingests high-volume data streams, enforces regulatory compliance, and serves real-time query APIs to carriers and regulators.

Data Scale

High-volume device registration pipelines ingesting data from multiple network operators

Performance

Query optimisation and indexing strategies to maintain low-latency reads on large datasets

Observability

Structured logging, metrics dashboards, and alerting to keep pipeline health visible

Deployment

Containerised services with automated CI/CD enabling confident, zero-downtime releases

Engineering challenges

  • ▸ Designing fault-tolerant data ingestion pipelines that handle upstream inconsistencies from multiple operators without data loss or silent corruption
  • ▸ Keeping query latency acceptable as dataset size grew — required deliberate schema evolution, targeted indexing, and query plan analysis
  • ▸ Building observability from scratch so on-call engineers could diagnose issues without needing deep system knowledge
  • ▸ Shipping infrastructure changes (Dockerisation, pipeline refactors) without disrupting live data flows or creating deployment risk

Team Leadership
